Encoding: value types to binary

This page will describe how values are encoded. For now it is summary information only.

Encoding Ints

A simple variable length integer encoding is used, similar to that in Perl (available using pack "w", $value) and in Google's ProtocolBuffer.

It differs from both mildly, as Google's format seems abiguous and confused, and Perl's does not support negative integers.

Encoding other Numeric quantities

These are all built on the integer formats. Architecture-dependent floats can be easily transformed into the underlying integers, through either directly accessing the bit fields of the floating point number, or using IEEE math.

Encoding strings

These are just encoded by length and with data inside. Strings will generally be UTF-8 otherwise they must be encoded as octets.
